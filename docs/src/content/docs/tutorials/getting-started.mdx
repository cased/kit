---
title: Getting Started with Kit
---

Welcome to **Kit**!  In this short guide you’ll:

1. Install Kit and its core dependencies.
2. Index a repository.
3. Run your first semantic code search.

> **Prerequisites**
>
> * Python ≥3.10
> * The [`uv`](https://github.com/astral-sh/uv) package manager installed (`pip install uv`).
> * An LLM API key (e.g. `OPENAI_API_KEY`) if you plan to generate summaries.

---

## 1  Install Kit

```bash
uv pip install kit[openai] sentence-transformers chromadb
```

---

## 2  Index your codebase

Create a small script `index_repo.py`:

```python
import kit
from sentence_transformers import SentenceTransformer

repo = kit.Repository("/path/to/your/project")
embed_fn = SentenceTransformer("all-MiniLM-L6-v2").encode

indexer = kit.DocstringIndexer(
    repo,
    repo.get_summarizer(),
    embed_fn=lambda txt: embed_fn(txt).tolist(),
)
indexer.build(level="file")  # one summary per file
```

Running the script will generate a `.kit/docstring_db` directory containing the vector index.

---

## 3  Search!

Add a quick search snippet to the end of `index_repo.py`:

```python
searcher = kit.SummarySearcher(indexer)
for hit in searcher.search("retry back-off logic", top_k=3):
    print(hit["file"], "→", hit["summary"])
```

You should see the most relevant files with concise summaries.

---

## Next steps

* Switch to **symbol-level** indexing: `indexer.build(level="symbol")`.
* Combine with exact keyword search: `repo.search_text("some_function")`.
* Explore the other tutorials (Docstring Search, AI PR Reviewer, etc.).
